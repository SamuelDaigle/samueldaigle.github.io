3D Game Engine Moteur de jeu 3D

I learned from multiple tutorials, books and articles about making a game engine. This project started during winter 2013 when I first used c++. I've built this engine multiple times, learning from my mistakes until I greatly improved my programming and architectural skills. At first, I used the DirectX SDK until I switched to Windows SDK. I use an XML file as a bridge between the c# user interface of the game engine and the c++ executable project. This experience will guide me in the design of my own engine and in the progress of futur games. Durant l'été 2015, je travaillais sur un jeu 3D utilisant Unreal Engine. Je me suis aussi beaucoup amusé avec le déplacement de l'eau afin de la rendre réaliste. Cette expérience va me guider dans la conception de mon propre engin et dans le progrès de mes futurs jeux.
